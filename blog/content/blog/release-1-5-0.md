## What's new?
- Migrated the database engine to SQLite, existing story data is migrated fully in the background without you having to do anything.
- Merged the Chat and Visual Novel themes.
- When editing text it is now more inline with iOS 26 iMessage editing.
- You can now set a custom profile picture and name for a message.
- Coins are no longer required to upload, search, and download media files, when using the Creator.
- When accessing your sessions you're prompted for biometrics first.

## What's fixed?
- Fixed an issue where downloading a cloud save would not show its media files.
- Fixed an issue where media files were not getting saved to the cloud automatically.
- Fixed an issue where media files were getting corrupted after saving to the cloud.
- Fixed an issue where moving the character sorting order would break the it.
- Fixed an issue where text would vanish if saved too quickly.
- Fixed an issue where the Story was be classified as sensitive, even though it is not.
- Fixed an issue where the sounds would not play or play incorrectly.
- Fixed an issue where the image editor would corrupt the image after saving.
- Fixed an issue where the image editor would not save the image after editing.
- Fixed an issue where the image editor would not apply the correct width of a edited image.
- Fixed an issue where the app would crash when playing certain videos.
- Fixed an issue where you could ”speedrun” the chat when using the enter button on the keyboard.
- Performance improvements for older and low end android devices.
