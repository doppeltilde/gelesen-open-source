Download the update: [gelesen.app](https://gelesen.app)

## What's changed?
- Gelesen is now available on Windows PCs through the [”Google Play Games on PC”](https://play.google.com/googleplaygames/?hl=en) store.
- On Android you can now log in with Google Play Games.
- Improvements were made to Gelesen when playing on a Apple silicon Mac.
- You can now set if you want to hide or show the ”Talking to Character” in Chat and Group Chat.

## What's fixed?
- Fixed an issue where the splash icon was grey instead of black when the device was in light mode.
- Fixed an issue when manually added media files to a media folder would not show them inside of the app.
- Fixed an issue where changes to scene settings would not be instantly reflected.
- Fixed an issue where the story would get stuck in a loop when updating the story.
- Fixed an issue where older stories would show a gray screen during playthrough.
- Fixed an issue where clear chat would result in a loop.
- Fixed an issue where fade would result in a loop.
