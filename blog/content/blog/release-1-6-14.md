## What's changed?
- You can now hide the ”Talking to Character” in Chat and Group Chat.
- Fixed an issue when manually added media files to a media folder would not show them inside of the app.
- Fixed an issue where changes to scene settings would not be instantly reflected.
- Fixed an issue where the story would get stuck in a loop when updating the story.
- Fixed an issue where older stories would show a gray screen during playthrough.
- Fixed an issue where clear chat would result in a loop.
- Fixed an issue where fade would result in a loop.
